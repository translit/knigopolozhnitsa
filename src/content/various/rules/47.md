# Є҆гѡ́ же а҆гаѳѡ́нꙋ е҆пї́скопꙋ

## Пра́вило г҃і.

<red>М</red>аѯі́мъ ѹ҆твержда́лъ, ꙗ҆́кѡ встꙋпи́лъ въ незако́нное сꙋпрꙋ́жество, по невѣ́дѣнїю зако́нѡвъ це́ркви: и҆ поели́кꙋ є҆го̀ смꙋща́етъ ѿлꙋче́нїе є҆гѡ̀ ѿ собра́нїй церко́вныхъ, то ѹ҆вѣрѧ́лъ, ꙗ҆́кѡ, ѹ҆чини́въ беззако́нїе по невѣ́дѣнїю, по согла́сїю возде́ржитсѧ ѿ беззако́ннагѡ сожи́тїѧ, на что̀ и҆ жена̀ є҆гѡ̀ согла́сна. И҆ та́къ а҆́ще ѹ҆смо́тришь, ꙗ҆́кѡ ѻ҆нѝ дѣ́лаютъ сїѐ по согла́сїю и҆ не ѡ҆бма́нываютъ: то, поели́кꙋ де́сѧть ѹ҆жѐ лѣ́тъ прошло̀ ѿ и҆́хъ сочета́нїѧ, а҆́ще заблагоразсꙋ́дишь поста́вити и҆̀хъ съ ѡ҆глаше́нными, ѹ҆стро́й та́кѡ. А҆́ще же примѣча́ешь, ꙗ҆́кѡ хотѧ́тъ ѡ҆бманꙋ́ти, и҆ потре́бна про́тивъ ни́хъ є҆щѐ стро́гость: сотворѝ, что̀ бг҃ъ тебѣ̀ внꙋши́тъ, рꙋково́дствꙋѧсь во все́мъ ѡ҆смотри́тельностїю. И҆́бо находѧ́сь на мѣ́стѣ, лꙋ́чше мо́жешь зна́ти расположє́нїѧ и҆́хъ.
